# TicTacToe

Implemented using python3, pygame, and flask.

![TicTacToeDemo](./assets/TicTacToeDemo.png)

## Getting Started

### Prerequisites
```
python3
```

### Installing
```
pip install -r requirements.txt
```

## Run Program
```
python api.py
python main.py
```
